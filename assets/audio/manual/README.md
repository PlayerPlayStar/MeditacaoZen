## Arquivos de Áudio Manuais ##

Essa pasta contém os arquivos de áudio que serão usados pelo sistema


## Como Usar

1. Adicione arquivos: Coloque arquivos MP3, WAV, OGG ou MP4 nas pastas
2. Prioridade: Arquivos manuais têm prioridade sobre sons sintéticos
3. Ordem: O sistema escolhe automaticamente o primeiro arquivo da pasta


## Formatos Suportados

- MP3, WAV, OGG, MP4
